COMO USAR

Clique no local do mapa desejado para criar um ponto. Para criar um ponto, é necessário inserir nome, descrição e classificação para ele. Para saber mais sobre as classificações, clique aqui.

Para alterar ou deletar um local criado, clique no ponto existente e utilize os botões de editar ou deletar. As mesmas regras utilizadas na criação são obrigatórias nas edições. Caso tenha deletado um local por engano, entre em contato conoscos para podermos restaurá-lo.

Para traçar uma rota até o local, clique no ponto e selecione a opção "Ir". Para a rota ser traçada automaticamente, é necessário dar permissão de localização. Caso não tenha dado a permissão, por favor, insira o endereço na janela ao topo do mapa.

LEGENDA

Este ícone representa um local totalmente acessível para um deficiente físico. É ideal para locais com rampas, elevadores, e acessos especiais cujo arredores também são acessíveis.

Este ícone representa um local parcialmente acessível para um deficiente físico. É ideal para locais com alguns itens de acessibilidade ou com arredores não apropriados para deficientes físicos.

Este ícone representa um local inacessível para um deficiente físico. É ideal para locais sem itens de acessibilidade e com arredores não apropriados para deficientes físicos.
